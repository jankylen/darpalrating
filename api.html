Public API Methods
  • api: http://api.darpalrating.org/api/index/get
  • method: GET
  • input parameters:
    • (int)indexType: query index type(1:BVIX,2:DPC)
    • (int)period: query time period (0:12 hour, 1:one day, 2:one week, 3:one month, 4:all time)
  • example url:
    • http://api.darpalrating.org/api/index/get?indexType=1&period=0
  • response:
    • latestValue: latest value
    • changeValue: recent 24hour changed value
    • changePercent: recent 24hour changed percent
    • isPositive: 1:increase ,2: decrease
    • indexValue: [timestamp,index value]
    • btcPrice: [timestamp, btc price]
    • ethPrice: [timestamp, eth price]
